# TextCasePro

TextCasePro is a simple, fast, and privacy-friendly text case converter that runs entirely in your browser.

It allows you to instantly convert text into different formats such as:

- UPPERCASE
- lowercase
- Title Case
- Sentence case

## 🚀 Features

- Instant text conversion
- No installation required
- Works directly in browser
- Clean and simple interface
- 100% private (no data stored)

## 💡 How it works

Simply paste your text into the input box and choose the format you want. The tool instantly converts your text without any delay.
